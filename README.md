# InTheMoment 🌿⏳

بوت تيليقرام بسيط، مهمّته الوحيدة أن يرسل لك بين الحين والآخر تذكيرًا
بأن تعيش هذه اللحظة، وبأننا جميعًا فانون. تذكيرات عشوائية (2-3 يوميًا)
في أوقات غير متوقعة، حتى لا تتحول إلى روتين تعتاد عليه وتتجاهله.

## كيف يعمل

- عند الاشتراك (`/start`)، ينضم المستخدم إلى قائمة المشتركين (تُحفظ في
  قاعدة بيانات SQLite محلية تبقى بعد إعادة تشغيل البوت).
- كل يوم عند الساعة 00:05 (بتوقيت `TZ_NAME`)، يختار البوت لكل مشترك
  عددًا عشوائيًا من التذكيرات (2 أو 3 افتراضيًا) في أوقات عشوائية موزّعة
  ضمن نافذة زمنية نهارية (9 صباحًا - 11 مساءً افتراضيًا)، بفاصل زمني لا
  يقل عن ساعة بين تذكير وآخر.
- عند إرسال كل تذكير، يختار البوت رسالة عشوائية من بنك رسائل (`bot/messages.py`)
  يمزج بين تذكيرات "تذكّر الفناء" (memento mori) وتذكيرات "الحضور في اللحظة"
  (mindfulness)، مع تجنّب تكرار نفس الرسالة مرتين متتاليتين لنفس الشخص.
- الاشتراك الجديد في منتصف اليوم يحصل فورًا على جدول تذكيرات للوقت
  المتبقي من نفس اليوم، فلا داعي للانتظار حتى الغد.

## الأوامر

| الأمر | الوظيفة |
|---|---|
| `/start` | الاشتراك في التذكيرات |
| `/now` | إرسال تذكير عشوائي فورًا (بدون التأثير على الجدول اليومي) |
| `/stop` | إلغاء الاشتراك وإيقاف كل التذكيرات القادمة |
| `/help` | عرض شرح مختصر عن البوت |

## التشغيل محليًا

يتطلب Python 3.11+.

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

cp .env.example .env
# افتح .env وضع فيه BOT_TOKEN الخاص بك (احصل عليه من @BotFather على تيليقرام)

python -m bot.main
```

## التشغيل عبر Docker

```bash
cp .env.example .env
# عدّل .env كما تريد
docker compose up -d --build
```

قاعدة البيانات تُحفظ في مجلد `./data` على الجهاز المضيف حتى تبقى بعد
إعادة بناء الحاوية.

## التشغيل كخدمة systemd (على خادم Linux)

1. انسخ المشروع إلى `/opt/inthemoment` وأنشئ بيئة افتراضية فيه (`venv`)
   ثم ثبّت المتطلبات، وأنشئ ملف `.env` بجانبها.
2. انسخ `deploy/inthemoment-bot.service` إلى `/etc/systemd/system/`.
3. فعّل الخدمة:

```bash
sudo systemctl daemon-reload
sudo systemctl enable --now inthemoment-bot
```

## التخصيص

كل إعدادات التوقيت موجودة في `.env` (انظر `.env.example` للتفاصيل):
عدد التذكيرات اليومية، النافذة الزمنية المسموح بها، الحد الأدنى للفاصل
بين تذكيرين، والمنطقة الزمنية.

لإضافة أو تعديل نصوص التذكيرات، عدّل القائمة `REMINDERS` في
`bot/messages.py` مباشرة — كل سطر هو رسالة مستقلة يختار البوت منها
عشوائيًا.

## ملاحظة

محتوى الرسائل مكتوب ليكون تأمليًا ومُلهمًا لا مقلقًا: تذكير بالفناء
هنا وسيلة لتقدير الحياة والحضور فيها، وليس دعوة للقلق. إن شعرت في أي
وقت أن التذكيرات تزعجك أكثر مما تفيدك، يمكنك ببساطة إرسال `/stop`.
