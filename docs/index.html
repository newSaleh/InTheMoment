🌿 إدارة اقتباسات بوت اللحظة

تصفّح، ابحث، عدّل، أضف، أو احذف اقتباسات البوت — والتعديلات تُرفع مباشرةً إلى GitHub من هذه الصفحة.

إعداد صلاحية الرفع (توكن GitHub)

بدون توكن يمكنك التصفح والبحث فقط. لرفع تعديلاتك تحتاج توكن بصلاحية كتابة على هذا المستودع تحديدًا:
1) اذهب إلى صفحة إنشاء Fine-grained token.
2) في "Repository access" اختر "Only select repositories" ثم اختر InTheMoment.
3) في "Permissions → Repository permissions" اختر "Contents: Read and write".
4) أنشئ التوكن وانسخه (يظهر مرة واحدة فقط) والصقه هنا.
يُحفظ التوكن في متصفحك فقط (localStorage) ولا يُرسل إلا مباشرة إلى GitHub API — لا تُشارك هذا التوكن مع أي أحد أو أي موقع آخر.
